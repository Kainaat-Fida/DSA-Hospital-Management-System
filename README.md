# DSA Hospital Management System

A Hospital Management System developed in C++ using Data Structures and Algorithms concepts.

## Features

- Doctor Management
- Patient Management
- Appointment Booking
- Stack Implementation using Linked List
- Sorting Doctors and Patients by Name
- Dijkstra Algorithm for Hospital Department Navigation
- Priority Queue Usage
- Object Oriented Programming

## DSA Concepts Used

- Linked List
- Stack
- Templates
- Sorting Algorithms
- Graphs
- Dijkstra Algorithm
- Priority Queue

## Technologies

- C++
- STL

## Run

Compile:

g++ main.cpp -o hospital

Run:

hospital.exe
